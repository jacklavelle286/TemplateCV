Welcome to my Static Website which I have hosted on S3!

How did I do it?

  1. First I created an S3 Bucket.
  2. Then I enabled static website hosting on the bucket.
  3. Third, I changed the public access settings.
  4. I then added the bucket policy which makes this html file publicly available.
  5. Then you choose this document as the 'index.html' document.
  6. Then we tested the website!
